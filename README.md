## AI Resume Builder Frontend
Frontend for users to upload resumes, analyze job descriptions, and view job matching results.

### Tech Stack
- React
- Tailwind CSS
- Axios (for API requests)
- React Router (for navigation)

### How to Run
1. Clone the repository:
   `git clone https://github.com/Hadylle/ai-resume-builder-frontend.git`
2. Install dependencies:
   `npm install`
3. Start the app:
   `npm start`
4. Configure `.env` to point to your backend API URL.
